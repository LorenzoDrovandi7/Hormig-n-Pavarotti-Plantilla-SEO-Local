# Hormig-n-Pavarotti-Plantilla-SEO-Local
Plantilla HTML, Tailwind CSS y JavaScript para sitios web de plantas de hormigón. Optimizada para SEO local, permite crear páginas por localidad dentro de un radio definido y facilitar la indexación en Google. Ideal para negocios de hormigón que quieran aparecer en búsquedas locales y generar leads.
